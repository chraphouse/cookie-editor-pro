# Cookie Editor Pro

A Chrome extension for viewing, editing, and managing cookies and localStorage. Save and restore cookie profiles, search across entries, and export data as JSON.

## Features

- View, edit, and delete cookies and localStorage entries
- Cookie profiles: save and restore sets of cookies per site
- JSON pretty-print for cookie and storage values
- Search and filter across all entries
- Export cookies and localStorage as JSON

## Installation

1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** and select this directory

## Built with

Chrome Manifest V3 (MV3)
