# Chrome Web Store Listing

## Name
Cookie Editor Pro

## Short Description (132 chars max)
View, edit, delete, and manage cookies & localStorage for any website. Cookie profiles for account switching. Modern MV3 extension.

## Detailed Description
Cookie Editor Pro is a powerful, modern cookie and storage manager for Chrome. Built from the ground up for Manifest V3.

**Features:**
- View all cookies for any website with detailed info (Secure, HttpOnly, SameSite, expiry)
- Edit cookie values, domains, paths, and all attributes
- Add new cookies with full control over all fields
- Delete individual cookies or clear all at once
- View and edit localStorage and sessionStorage
- JSON values are automatically pretty-printed for easy editing
- Search and filter through all cookies and storage items
- Export cookies/storage as JSON files
- Import cookies/storage from JSON
- Cookie Profiles — save and restore complete cookie sets per site (great for switching between accounts!)
- Storage size indicators for localStorage items
- Session vs persistent cookie indicators
- Expired cookie warnings
- Dark theme with clean, modern UI
- Zero permissions beyond what's needed — no tracking, no analytics

**Why Cookie Editor Pro?**
After EditThisCookie was removed from the Chrome Web Store, developers and testers needed a modern replacement. Cookie Editor Pro goes beyond cookies — it also manages localStorage and sessionStorage, and adds cookie profiles for easy account switching.

**Perfect for:**
- Web developers debugging cookies and storage
- QA testers switching between test accounts
- Privacy-conscious users managing site data
- Anyone who needs to view, edit, or export cookies
